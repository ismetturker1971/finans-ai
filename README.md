# Finans AI - Türkçe Finansal Analiz Platformu

Claude AI tarafından desteklenen profesyonel finansal analiz platformu. 10 farklı finansal analiz türünü yönetim seviyesinde raporlar halinde sunar.

## 🚀 Özellikler

- **10 Finansal Analiz Türü**
  - 📊 Hızlı Finansal Özet
  - 📈 Bütçe-Gerçekleşen Sapma Analizi
  - 💰 Nakit Akışı ve Pozisyon Takibi
  - ⏰ Alacak/Borç Yaşlandırma
  - 💹 Kârlılık Analizi
  - 📉 Gider Analizi ve Anomali Tespiti
  - 🏛️ KDV Yükü ve Nakit Etkisi
  - 💼 Vergi Yükü Tahmini
  - 💱 Kur Farkı ve Döviz Pozisyonu
  - 📐 Enflasyon Muhasebesi Etkisi

- **Türkçe Arayüz** - Tam Türkçe destek
- **Claude AI Entegrasyonu** - Profesyonel analiz
- **Hızlı & Güvenli** - Verileriniz işlenmez, kaydedilmez
- **Responsive Design** - Mobil & Desktop

## 📋 Ön Koşullar

- Node.js 18+
- npm veya yarn
- Anthropic Claude API Anahtarı

## 🛠️ Kurulum

### Yerel Geliştirme

```bash
git clone https://github.com/ismetturker1971/finans-ai.git
cd finans-ai
npm install
echo "CLAUDE_API_KEY=sk-ant-..." > .env.local
npm run dev
```

### Vercel'e Deployment

1. https://vercel.com adresine gidin
2. GitHub hesabınızı bağlayın
3. `finans-ai` repository'sini seçin
4. Environment Variables'a `CLAUDE_API_KEY` ekleyin
5. Deploy tıklayın

Detaylı rehber: Bkz [SETUP.md](./SETUP.md)

## 📖 Kullanım

1. Analiz türü seçin
2. Finansal veri yapıştırın
3. "✨ Analiz Et" tıklayın
4. Sonuçları görün

## ⚠️ Önemli Uyarılar

- Analizler yönetim seviyesi karar desteği için hazırlanmıştır
- Resmi beyan, e-defter yerine geçmez
- Vergi konularında mali müşavire danışınız
- API anahtarını güvenli saklayın
